## üéØ What Are Ecosystem Statistics?

Ecosystem Statistics replace the old "Project Statistics" section to show you:

- **Total ecosystem size** across all marketplaces
- **Growth trends** over time
- **Developer community** health
- **Plugin quality** indicators
- **Category distribution** and trends

Unlike traditional GitHub repository statistics that show data for a single project, ecosystem statistics aggregate data from multiple Claude Code marketplaces and plugin repositories.

## üÜò Getting Help

### Common Questions

**Q: How often is data updated?**
A: Ecosystem data is refreshed every 6 hours with real-time options available.

**Q: Why do some plugins show as "unverified"?**
A: New plugins undergo verification process. This typically takes 24-48 hours.

**Q: Can I export the data for my own analysis?**
A: Yes! Use the export button to download data in CSV or JSON format.

**Q: What does the health score mean?**
A: It's a composite metric (0-100) considering verification rates, maintenance, quality scores, and security.

**Q: How are growth trends calculated?**
A: Based on plugin counts, developer additions, and download activity across all marketplaces.
